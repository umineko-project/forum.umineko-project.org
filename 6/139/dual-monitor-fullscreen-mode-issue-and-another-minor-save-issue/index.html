I discovered that if the game is running in auto mode and I click to save a file, when I load that file, a green arrow would remain on the upper left corner, which is the indicator for auto mode. But that's misleading because we must have interrupted the auto play in order to save and load a file. So the game would just continue in manual mode, with the misleading green arrow shown at the corner. The arrow would go away if I press A to enter auto mode and press A again to cancel it.
