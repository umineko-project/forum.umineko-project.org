personally, I use this code to delay the final phrase of a voice so that if it's very short it gives me some time to read it.

file: [ons.cfg] add this text:
final-voicedelay-time=1600

I use this parameters:
voicedelay-time=500
final-voicedelay-time=1600

So the dialogs are not very rushed.
Tell us if something worked!

PD: Also, how fast the text appears, just push "2" (normal, I use it) o "1" (slow, but I think it's too slow), so if even the voices are faster than the text, until the text reaches the final line, it won't skip to the next dialog.
