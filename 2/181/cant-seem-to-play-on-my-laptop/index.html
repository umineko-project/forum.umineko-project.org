Hi,
This error message implies that you do not have the latest/correct graphics drivers installed.
The game does not support software rendering, because this way it will be very slow, and neither it does support DirectX, because it has a terrible API and is not available on macOS, which is the main target platform.
Unfortunately sometimes it is not easy find the working video drivers for laptops, but without the hardware info I could not help you.
